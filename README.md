# Document Tampering and Forgery Detection

A computer vision project for detecting forged or tampered regions in document images.

The goal is to build a prototype that can identify suspicious modifications such as edited text, changed fields, copied regions, or replaced information, and visualize the suspicious area on the document.

## Current Status

Project vision and technical scope are being finalized.

## Planned Direction

Possible approach:

- Generate or collect document images
- Create authentic and tampered samples
- Train a detector to localize tampered regions
- Visualize predictions using bounding boxes
- Evaluate using detection metrics such as precision, recall, and mAP
